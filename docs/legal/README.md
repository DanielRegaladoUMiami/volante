## The core problem (one paragraph)
Volante's driver operates the **customer's own car** for a fee. That breaks the Florida TNC model:
(1) the customer's personal-auto **livery/"for-hire" exclusion** can void their coverage; (2) Florida's
**dangerous-instrumentality doctrine** makes the customer-owner vicariously liable; and (3) Fla. Stat.
**§ 627.748 (TNC) likely doesn't apply** (a "TNC vehicle" must be owned/leased/authorized-to-be-used by
the *driver*), which removes state preemption and likely puts us under **Miami-Dade Code Ch. 31
(Vehicles for Hire)**.
